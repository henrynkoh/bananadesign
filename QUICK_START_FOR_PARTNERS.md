### Option 1: Deploy to Vercel (2 Minutes - Recommended)

1. **Go to:** https://vercel.com
2. **Sign up/Login** with your GitHub account
3. **Click:** "New Project"
4. **Import:** `henrynkoh/bananadesign`
5. **Add Environment Variable:**
   - Name: `REPLICATE_API_TOKEN`
   - Value: [Get free token from https://replicate.com]
6. **Click:** "Deploy"
7. **Done!** You'll get a live URL like: `https://bananadesign.vercel.app`

**That's it!** The site will be live and accessible to anyone.

---

### Option 2: Run Locally (5 Minutes)

```bash
# 1. Clone the repository
git clone https://github.com/henrynkoh/bananadesign.git

# 2. Go into the folder
cd bananadesign

# 3. Install dependencies
npm install

# 4. Create environment file
echo "REPLICATE_API_TOKEN=your_token_here" > .env.local

# 5. Run the development server
npm run dev

# 6. Open your browser to:
# http://localhost:3000
```

**Note:** You'll need Node.js installed. Get it from: https://nodejs.org

---

1. **Go to:** https://github.com/henrynkoh/bananadesign
2. **Browse files** to see the code
3. **Read README.md** for overview
4. **Check components/LandingPage.tsx** for landing page
5. **Review documentation** in root folder
